# webpack

### 문제

    ```jsx
    app.get('/project/:id', checkLogin, function (req, res) { res.render('index.html') })
    ```

express 위와 같이 라우터를 설정 후 `host/project/1` 의 url로 접근 했을 경우 `Uncaught SyntaxError: Unexpected token <`와 같은 에러가 발생했다.

### 해결

```html
<base href="/" />
```
